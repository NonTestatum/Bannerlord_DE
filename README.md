

### Installation Helper:
`npm install`

### Verwendung Helper:
`node . --getFiles`

Holen der Sprachdateien aus dem Spieleverzeichnis (nach Update für Vergleich)

`node . --compare`

Überprüfen und vergleichen der xml Einträge von Original und Übersetzung
